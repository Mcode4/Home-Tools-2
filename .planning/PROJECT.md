# Project: Home Tools v2

## Mission
To build a professional-grade Property Inspection and Documentation suite that enables rapid capture of geospatial property data, 2D/3D room rendering, and high-fidelity reporting for home inspectors and real estate professionals.

## Core Pillars
1. **Precision Capture**: High-end map editor for pinning properties, rooms, and technical points.
2. **Immersive Visualization**: Transitioning from 2D vector room outlines to 3D item/storage representations.
3. **Professional Deliverables**: Automated generation of JSON and PDF inspection reports.
4. **Premium UX**: A seamless, dashboard-driven experience with high-fidelity UI aesthetics.

## Tech Stack
- **Frontend**: React, Redux, MapLibre GL, CSS (Vanilla/Premium)
- **Backend**: Python (FastAPI), SQLite/PostgreSQL, SQLAlchemy
- **Infrastructure**: Docker, Nginx

## Success Criteria
- [ ] Fully functional Property Editor with sidebar-driven detail management.
- [ ] Robust Render Engine for 2D room layouts and 3D item storage.
- [ ] Professional Export System for JSON/PDF report generation.
- [ ] Stabilized and polished dashboard, profile, and error pages.
